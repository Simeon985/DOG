# LED Emotion Recognition — User Observation Notes

## Context

Test session to evaluate how well emotions displayed on 8×8 LED matrix eyes are recognized.  
Different emotion patterns were shown in both **small and larger versions**.

---

## Observations

### 1. Heart Pattern (Large)

- Initial interpretation: apple or heart (after hesitation → heart)
- Perceived emotion: surprised / needy (e.g. thirsty)
- Remark: eyes themselves were not very noticeable

---

### 2. Sad Pattern (Large)

- Interpretation: someone looking downward
- Perceived emotion: sadness
- Remark: emotion was clearly recognizable

---

### 3. Angry Pattern (Standard Size)

- Emotion became much clearer when both eyes were perfectly aligned
- Perceived emotion: anger
- Remark: could benefit from an additional “extra angry” stage

---

### 4. Neutral Pattern

- Perceived emotion: curiosity / observing
- Remark: good candidate for animation

---

### 5. Sad Pattern (Larger)

- Emotion unclear to the user
- Recognition confidence was low

---

### 6. Angry Pattern (Larger)

- Perceived emotion: anger
- Remark: previous (smaller) angry pattern was clearer

---

### 7. Neutral Pattern (Larger)

- Interpretation: guilt / sadness / looking down
- Remark: larger version conveyed a different emotion compared to the smaller version

---

### 8. Size Awareness Effect

- User realized patterns were simply scaled versions
- Larger heart shapes were more easily recognized
- User also mentioned this version looked more like hearts than the previous one (some contradiction in feedback)

---

## Key Takeaways

- **Eye alignment strongly affects emotion recognition**
- **Scaling patterns changes perceived emotion**
- Larger symbols improve recognizability (especially hearts)
- Some emotions (strong sadness variants) remain ambiguous
- Animation potential is high for curiosity-type expressions
